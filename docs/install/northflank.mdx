Deploy Durar on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy Durar](https://northflank.com/stacks/deploy-Durar) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy Durar now**.
4. Set the required environment variable: `Durar_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the Durar template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the Durar service.
8. Open the public Durar URL at `/Durar` and connect using your `Durar_GATEWAY_TOKEN`.

## What you get

- Hosted Durar Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so config/credentials/workspace survive redeploys

## Connect a channel
