# `Durar tasks flow`

Flow commands are subcommands of `Durar tasks`, not a standalone `flows` command.

```bash
Durar tasks flow list [--json]
Durar tasks flow show <lookup>
Durar tasks flow cancel <lookup>
```

For full documentation see [Task Flow](/automation/taskflow) and the [tasks CLI reference](/cli/index#tasks).
